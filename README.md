# OfficeAI Suite – Local AI Add-ins for Word, Excel and Outlook

OfficeAI Suite è una raccolta di estensioni locali per Microsoft Office
(Word, Excel, Outlook) che integrano Ollama come motore AI locale.

## Add-ins inclusi
- **Word Add-in** (generazione contenuti, riassunti)
- **Excel Add-in** (analisi testo, funzioni AI integrate)
- **Outlook Add-in** (generazione email professionali, riassunti conversazioni)

## Requisiti
- Node.js
- Yeoman Office Generator
- Ollama installato (LLAMA3 8B consigliato)
- Modello AI configurato nel backend locale
